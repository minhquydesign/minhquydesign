# cơ bản

+ nhập

- MD ___ TẠO THƯ MỤC (make directory)
- CD ___ TRUY CẬP THƯ MỤC ( change directory )
- .
- ..
- DEL
- MOVE
- rd  remove directory ( xoá thư mục )
